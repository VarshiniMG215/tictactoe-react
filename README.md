Premium Tic-Tac-Toe (React)
I built a  beginner level game  using  javascript+react !
A modern, responsive Tic-Tac-Toe game built using React.js. This project focuses on clean state management, modular component structure, and a polished "Dark Mode" user experience.

**Key Features**
Dynamic State Management: Real-time tracking of player turns (X and O) and game board updates using React Hooks.

Winner Detection Algorithm: Implemented logic to identify winning combinations and draw scenarios instantly.

Modern UI/UX: Styled with a dark-theme aesthetic, featuring CSS animations for "winning" highlights and smooth transitions.

Responsive Design: Fully functional across desktop, tablet, and mobile browsers.

**Tech Stack i used **
Frontend: React.js (via CDN for lightweight deployment)

Styling: CSS3 (Flexbox/Grid, Custom Animations)

Logic: JavaScript 

**Project Structure**
index.html - The entry point and React root container.

script.js - Contains the React functional components and game logic.

style.css - Custom styles and responsive layout definitions.

** What I Learned **
During this project, I strengthened my understanding of:

React Hooks (useState): Managing the game board and turn-based logic effectively.

Conditional Rendering: Displaying game status (Winner, Draw, or Next Player) dynamically.

Array Mapping: Using .map() to render the game grid cleanly from a single state array
